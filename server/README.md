# personal-finance-manager

A full-stack MERN application for managing personal finances, tracking income and expenses, setting budgets, and visualizing spending trends.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- Chart.js

## Features
- User Authentication
- Income & Expense Tracking
- Budget Management
- Financial Dashboard
- Monthly Reports
- Responsive Design

## Project Structure
```
personal-finance-manager
├── client
└── server
```

## Installation

### Backend
```bash
cd server
npm install
npm run dev
```

### Environment Variables
Create a `.env` file:

```env
PORT=5000
MONGO_URI=
JWT_SECRET=
JWT_EXPIRES_IN=7d
```
